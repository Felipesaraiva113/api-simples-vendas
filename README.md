Este projeto marca minha primeira implementação de uma API RESTful, desenvolvida para automatizar a leitura de grandes volumes de dados e fornecer resultados processados em tempo real. Utilizando a biblioteca Pandas para manipulação de DataFrames e o microframework Flask, a API realiza a extração de dados de planilhas e retorna o cálculo total de vendas via endpoint JSON. Diferencial: Demonstra capacidade de integrar ferramentas de Ciência de Dados com desenvolvimento web, focando em soluções práticas para Business Intelligence e automação de relatórios corporativos.
